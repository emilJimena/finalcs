## 🏗️ Architecture Overview

```
HTTP Request
    ↓
public/index.php (Front Controller)
    ↓
Router (matches URL to route)
    ↓
Controller (processes request)
    ↓
Repository (queries database)
    ↓
Entity (data model)
    ↓
Template (renders response)
    ↓
HTTP Response
```
